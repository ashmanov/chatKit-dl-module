# notificationDisplay

## Description
Отображение определенной нотификации

## Typical data
```
NotificationDisplayData {   
  messageId: number   
}}   
```

## Example of data
```
 data = {   
   messageId: 12,   
}
   ```
   
## Example of call of method
```
moduleDispatcher('notificationDisplay',{ messageId: 12})
```
